Visão Geral - Proposta

Esse é um projeto para desenvolvimento de um jogo. Esse jogo tem uma estrutura de fases com objetivos claros: chegar do ponto de partida ao ponto final. Durante a fase o jogador deve resolver um “quebra-cabeça” para abrir caminho até seu objetivo. As fases colocam o jogador no ponto mais baixo com o objetivo dele chegar ao ponto mais alto. Deve-se usar os blocos do cenário para criar uma escada. Assim, quais blocos usar e como usá-los é “quebra-cabeça”. Há dois principais tipos de blocos, ambos podem apoiar o jogador e outros blocos: blocos fixos: não são sujeitos a gravidade e não podem ser reposicionados; blocos móveis: sujeitos a gravidade se “atualizados” e podem ser reposicionados; O jogo tem uma estrutura 3D. Isso é, permite que o jogador se mova nas quatro direções (para frente, para trás, para direita e esquerda) e interaja com os blocos em diversas formas (empurrar, subir e se pendurar nas bordas).

Jogo referência - Rapunzel (Catherine)

O jogo que iremos desenvolver tem grande inspiração na mecânica do mini-jogo “Rapunzel” do jogo “Catherine”, mas para esse projeto planejo incluir e desenvolver outras mecânicas como blocos especiais e outras variações no jogo (mais personagens e opção para controlar mais de um player).
